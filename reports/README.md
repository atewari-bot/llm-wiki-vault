# Reports

Human-facing outputs generated by Claude Code on request.

| File pattern | Generated by |
|-------------|-------------|
| daily/YYYY-MM-DD.md | `daily` — morning briefing with Slack + Calendar |
| weekly/YYYY-WNN.md | `weekly` — week in review with Slack stats |
| onepager/<slug>.md | `onepager <topic>` — shareable topic summary |
| slides/<slug>.md | `slides <topic>` — Marp presentation deck |
| mindmap/<slug>.md | `mindmap <topic>` — Mermaid mind map |
| discoveries/YYYY-MM-DD.md | `discover` — hidden connections report |
| review/YYYY-MM-DD.md | `review` — confidence decay audit |

None of these files use wikilinks — they are written for human reading.
